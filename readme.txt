== Description ==
openFeed is a real time feeds service with analytics for you to discover and monitor what is trending online. 


Features:

* Real time tracking service, find stories when they break.
* Follow topics by author's name, domain name, category, and/or tags and get alerted when something new is published.
* See what articles are trending online from the past 30 minutes to 24 hours.


Available in these languages:

* English 



If you discover a problem with this plugin, please report it at https://wordpress.org/tags/openfeed


For further information and or API development, visit
https://www.openfeed.io/api-documentation/

== Installation ==

1. Download
2. Upload to your `/wp-contents/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
